## Examples: 
* `Boards/'board name'/Examples`

## Compile
```
cd Boards/'board name'/Examples/'example name'
make
```

## Directory Layout
**`Drivers`**: Shared drivers and libraries for all example projects.

**`Boards`**: Collection of Board specific examples.
